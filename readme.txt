== Description ==

Add a VAT calculator to your WordPress site, using the <a href="http://www.vatcalculatorplus.com">VAT Calculator Plus</a> web site widget.  You can set a fixed VAT rate or let your visitor set the rate.  The VAT calculator has a clean, modern look, and if you are a CSS guru, you can style it totally yourself.

Use the VAT Calculator Plus *Settings* page to generate a ShortCode that you can copy and paste into your content.

== Installation ==

1. Upload `vat-calculator-plus` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the *Plugins* menu in WordPress.
1. Add the shortcode to a post or page.
